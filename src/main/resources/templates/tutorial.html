Criando projeto no Quartus II

Com a Interface do quartus II aberta clique em File > New Project Wizard > Next.

Tela de escolha de diretório e nome

Escolha o diretório que ficará os arquivos do projeto, nome do projeto e o nome da Entidade de alto nível (recomenda-se o uso do mesmo nome do projeto).

Clique em Next > Next.


Tela de escolha de modelo de FPGA

Selecione os mesmos campos apresentados na imagem de forma que:

Em Device Family escolha Cyclone II.

Em Avaiable Devices escolha EP2C35F672C6.

(Dica): Em Name Filter digite 672 para encontrar mais rapidamente o dispositivo

Clique em Next > Next > Finish.


Adicionando biblioteca do VDLAB

Faça o download da biblioteca do VDLAB.

Descompacte os arquivos da biblioteca na pasta raiz do seu projeto.


Utilizando biblioteca do VDLAB

Crie um arquivo do tipo Block Diagram/Schematic File indo em:

File > New... > Block Diagram/Schematic File.


Tela de escolha do tipo de arquivo

Clique em Symbol Tool .

No menu de Libraries entre na pasta Project.

Escolha o modulo_inputs_outputs como na foto.


Selecione o componente mostrado na imagem

Pronto! agora basta conectar os inputs desejados do seu projeto no switch button do modulo de inputs virtuais, como usado na imagem abaixo.
