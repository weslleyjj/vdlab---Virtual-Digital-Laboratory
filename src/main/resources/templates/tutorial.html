Criando projeto no Quartus II

Com a Interface do quartus II aberta clique em File > New Project Wizard > Next.

Tela de escolha de diretório e nome

Escolha o diretório que ficará os arquivos do projeto, nome do projeto e o nome da Entidade de alto nível (recomenda-se o uso do mesmo nome do projeto).

Clique em Next > Next.


Tela de escolha de modelo de FPGA

Selecione os mesmos campos apresentados na imagem de forma que:

Em Device Family escolha Cyclone II.

Em Avaiable Devices escolha EP2C35F672C6.

(Dica): Em Name Filter digite 672 para encontrar mais rapidamente o dispositivo

Clique em Next > Next > Finish.


Adicionando biblioteca do VDLAB

Faça o download da biblioteca do VDLAB.

Descompacte os arquivos da biblioteca na pasta raiz do seu projeto.


Utilizando biblioteca do VDLAB

Crie um arquivo do tipo Block Diagram/Schematic File indo em:

File > New... > Block Diagram/Schematic File.


Tela de escolha do tipo de arquivo

Clique em Symbol Tool .

No menu de Libraries entre na pasta Project.

Escolha o modulo_inputs_outputs como na foto.


Selecione o componente mostrado na imagem

Pronto! agora basta conectar os inputs desejados do seu projeto no switch button do modulo de inputs virtuais, como usado na imagem abaixo.


Projeto utilizando a biblioteca do VDLAB

Pinagem de outputs para utilizar o VDLAB

Atenção!

Para escolher os pinos de saída do projeto faz se necessário seguir o padrão de nomenclatura.

Por exemplo: para o primeiro led vermelho da placa, nomear o outuput como LEDR[0]

Para o primeiro led verde da placa, LEDG[0]

Para displays de 7 segmentos HEX[6..0] e assim por diante

Pinagem pela biblioteca do VDLAB

Com os nomes dos outputs seguindo o padrão de nomenclatura correto e com o projeto compilado vá em:

Assignments > Import Assignments > ...

Selecione o arquivo de2.qsf na raiz do projeto. Arquivo provido pela biblioteca do VDLAB

Dê o OK.

Agora o Quartus II fará a atribuição dos pinos de cada output automaticamente.

Pinagem pelo Pin Planner do Quartus

Caso não possua o pdf com a localização dos pinos faça o Download aqui

Na tela do Quartus vá em:

Assignments > Pin Planner. Você verá a seguinte tela:


Interface do Pin Planner

Repare que no menu All Pins na parte mais baixa da tela, existe uma listagem com os outputs e inputs adicionados ao seu projeto.

Clique na célula pertencente a coluna Location e de acordo com oque preferir insira o pino desejado para o input ou output do projeto.

Após a escolha dos pinos de cada input/output do projeto re-compile seu projeto.

Após a correta definição dos pinos sempre compile seu projeto!

Rodando seu projeto no VDLAB

Na página inicial do VDLAB, clique no botão Procurar...

Navegue para a pasta do seu projeto do quartus e escolha o arquivo terminado em .sof

Caso não encontre, verifique a pasta output_files, se ainda não encontrar, compile seu projeto e procure o arquivo novamente.

Após escolher o arquivo clique em Enviar e acompanhar projeto

Utilize dos switch buttons e dos push buttons no painel de controle para testar seu projeto.

Divirta-se!
